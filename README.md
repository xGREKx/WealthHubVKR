# Wealth Hub

Платформа для привлечения и управления финансированием в малом и среднем бизнесе. Реализация по ВКР (Финансовый университет при Правительстве РФ, 2026).

Три роли: **Инвестор**, **Предприниматель**, **Администратор**. Соответствие ФЗ-152, ФЗ-259, ФЗ-115.

> **ВКР:** Верховский Григорий Андреевич, группа ИТвСФТ24-1м, 2026.

## 🚀 Быстрый старт

### 1. Backend (Django + DRF + SQLite)

```bash
cd backend

# Виртуальное окружение
python -m venv .venv
.venv\Scripts\activate              # Windows
# source .venv/bin/activate         # Linux / macOS

# Зависимости
pip install -r requirements.txt

# Миграции
python manage.py makemigrations core
python manage.py migrate

# Стартовые данные + демо-инвестор + дивиденды для проверки графиков
python manage.py seed_data --reset

# Суперпользователь для админки
python manage.py createsuperuser

# Запуск
python manage.py runserver
```

API: **http://localhost:8000/api/**
Admin: **http://localhost:8000/admin/**

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Фронт: **http://localhost:5173**

- Кнопка «Инвестор» → создаст случайного ЕСИА-инвестора (без истории)

- Кнопка «Администратор» → создаст ЕСИА-админа с правами модерации

> Для проверки графиков **войдите по логину `demo_investor`** — у него заполнены инвестиции, дивиденды и история транзакций. Иначе графики будут пустыми (это правильное поведение, в т.ч. для свежесозданного юзера).

## 🧪 Автоматические тесты

Покрывают рекомендательную подсистему (`core/recommender.py`), её HTTP-эндпоинт `/api/projects/<id>/recommendation/` и smoke-проверку эталонного демо-проекта **«InvestFlow PRO»**, который должен после `seed_data` существовать на модерации, оцениваться системой как `medium risk` и быть `high attractiveness` для `demo_investor`.

### Быстрый запуск

```bash
cd backend
.venv\Scripts\activate              # Windows
# source .venv/bin/activate         # Linux / macOS

# Все 39 тестов
python manage.py test core.tests

# С детальным выводом по каждому тесту
python manage.py test core.tests -v 2

# Только один модуль
python manage.py test core.tests.test_recommender
python manage.py test core.tests.test_recommendation_api
python manage.py test core.tests.test_seed_demo_project

# Параллельный запуск (быстрее на многоядерных машинах)
python manage.py test core.tests --parallel 4
```

Ожидаемый результат:
```
Ran 39 tests in ~17s
OK
```

## ✅ Готово в текущей сборке

### Фаза 1
- Реальная регистрация (Django + JWT) с выбором роли
- Кнопка Госуслуг в фирменном синем стиле
- Кнопки Войти/Зарегистрироваться скрываются на странице login
- Поиск выезжает справа из-под кнопки с анимацией
- Главная: «Горячие предложения» с умным таймером (5с авто, 15с после ручного клика)
- Marketplace с расширенными фильтрами + пагинация
- Личный кабинет с подтверждением через Госуслуги, аватаркой
- Финансы с агрегацией нескольких инвестиций в один проект
- Все графики на реальных данных + тултипы (donut, bar, line, heatmap)
- Тепловая матрица показывает только проекты ИЗ ПОРТФЕЛЯ
- Карусель донат-графиков в кабинете
- Вертикальный таймлайн дивидендов
- Вывод средств с расчётом комиссии и НДФЛ
- Поддержка с чатом, FAQ, Соцсети с дисклеймером Meta
- Settings, Footer с подписью Верховский Г. А.
- Корневой URL возвращает JSON со списком endpoints

### Фаза 2 (приоритеты)
- ✅ **Создание/редактирование проекта** на отдельной странице с обложкой, командой (включая загрузку CV в PDF), документами
- ✅ **Полная админ-панель**: модерация с просмотром деталей проекта, командой, документами; админ может скачать CV/документы; одобрить с указанием риска и привлекательности или отклонить с обязательным комментарием
- ✅ **Обновление портфеля после инвестиции**: после успешной сделки счётчик `portfolioVersion` инкрементируется → CabinetPage / FinancePage / AnalyticsPage перезагружают свои данные
- ✅ **Mock-данные для тестирования**: команда `seed_data --reset` создаёт демо-инвестора с 7 инвестициями (low/medium/high риск), дивидендами за 8 месяцев и историей транзакций — графики заполняются полноценно
- ✅ **Рекомендательная подсистема**: модуль `core/recommender.py` оценивает риск проекта по 7 параметрам и привлекательность для конкретного инвестора по 7 факторам. На модерации админу выводится виджет с предложением и breakdown по факторам; на витрине рекомендованные проекты получают фиолетовую рамку и бейдж «рекомендуем», уровень «Высокая» в карточке. Покрыто 39 автотестами.
- ✅ **Эталонный проект на модерации**: `seed_data` создаёт `InvestFlow PRO` (B2B SaaS для семейных офисов) — реалистичный проект с командой, документами и полным описанием, который рекомендатель оценивает как **средний риск** и **высокую привлекательность** для `demo_investor`.
